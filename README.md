# Sistema de Boletos Aéreos ✈️

## Descripción
Sistema de gestión de reservaciones de vuelos desarrollado en C# Windows Forms con MySQL.

## Características
- ✅ Gestión de usuarios
- ✅ CRUD de aerolíneas
- ✅ Gestión de vuelos y rutas
- ✅ Reservaciones de pasajeros
- ✅ Cálculo automático de tarifas (equipaje)
- ✅ Sistema de pagos
- ✅ Cancelaciones

## Tecnologías
- C# .NET Framework
- Windows Forms
- MySQL
- MySql.Data

## Instalación
1. Clonar el repositorio
2. Restaurar la base de datos MySQL con el script incluido
3. Configurar la cadena de conexión
4. Compilar y ejecutar
